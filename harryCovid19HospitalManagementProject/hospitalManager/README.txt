Java Spring Boot RESTFUL CRUD Application for hospital management system for covid 19 patients

This entire folder needs to be downloaded and copied inside the eclipse or STS workspace in order for the Maven build to execute

The Maven project should be updated after import by right clicking the project folder -> Maven -> Update Project

MySQL used as DBMS (code of mysql file found inside the projectInstructions in this folder, need to be coded in SQL Query Browser)

Sweet alert files should be imported inside the static -> css and js folders respectively(found inside the projectIntructions -> SweetAlert files)
an entire file structure given in the image inside projectInstructions labeled "project file structure"
